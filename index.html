What business problem will it solve?

Internal Tooling

  • Developer productivity in RWD is paramount
  • Reduce amount of re-work, re-invention between projects
  • Increase potential mobility between projects (a necessary evil)
  • Promote adoption of development standards and conventions across team
  • Provide a more stable and standardized dev environment

Socialization

  • Write clear, easily digestable documentation
  • Isolate from project-specific IP so that library code may be open-sourced
  • Democratize supported features by encouraging contributions
  • Establish EPAM's thought leadership in this area

What technical competency will we gain?

Node.js

  • Based on JavaScript language, familiar to team
  • Simplify external dependencies and platform idiosyncrasies
  • Modular framework enables high potential for extension
  • Express+Swig web stack can export to static HTML

Yeoman/Grunt

  • Sophisticated yet simple build system and project generator
  • Allows project-specific configuration as well as standard build process
  • Allows creation of generator templates
  • Uses robust plugin ecosystem

JavaScript Architecture

  • Bar has been raised considerably with client-side JS
  • Performance profiling and benchmarks across browsers & devices
  • Provide standard modular architecture conventions
  • Stay project-agnostic while delivering useful hooks
  • Reduce external library dependencies where possible

Modular UX Pattern Authoring

  • Acknowledge that a one-size-fits-all approach doesn't work
  • Provide customizable UX patterns that cover the 80% case by default
  • Curate a collection of common UX patterns that can be easily adapted to any project

How will this instill confidence in clients?

Responsive XD experts

  • Modern toolkit assembled from the ground up specifically for RWD
  • Best practices for client-side implementation across browsers & devices
  • Documentation and best practices consistent with the current device ecosystem

RWD Community

  • Soliciting and responding to rigorous community feedback drives continuous improvement
  • Increases visibility and reputation of our RWD competency
  • Shows our commitment to excellence for all aspects of RWD
